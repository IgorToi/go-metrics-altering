## «Metrics and alerting collection service»

### Requirements

PostgreSQL installed

Go installed

## Usage

### To run server, execute:

```shell
go run cmd/server/main.go
```

### To run agent, execute:

```shell
go run cmd/agent/agent.go
```

#### To run tests, execute:

<!-- start:code block -->

```shell
go test ./...
```
